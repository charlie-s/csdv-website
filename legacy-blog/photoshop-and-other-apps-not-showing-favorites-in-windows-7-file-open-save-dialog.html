Photoshop and other apps not showing favorites in Windows 7 file open / save dialog?

Thankfully I ran into this post: http://www.sevenforums.com/general-discussion/16988-changing-common-places-favorites-save-dialog.html

Adobe's CS4 (and definitely earlier) products are among the bandwagon that don't use the native Windows file open/save dialog. I'm not sure on why it's showing the older dialog, as other applications that aren't aware they're running in Windows 7 seem to show the new dialog. Anyways, the big problem here is that your places, shortcuts, favorites, bookmarks, whatever the hell you want to call them, aren't visible.

If you, like me, have setup some serious favorites in your "places" column (left side) of the explorer window, you will definitely come to rely on them for your workflow. In order to get these to display quickly in non-native file browsers, you will need to change some Group Policy permissions (gpedit.msc). If you've got Windows 7 Home premium (or lower) you won't be able to run gpedit.msc... unfortunately it's one of the features in business and ultimate.
